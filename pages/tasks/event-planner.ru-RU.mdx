# Техническое задание "Планировщик мероприятий"

## Описание

В рамках проекта необходимо разработать приложение для управления событиями, включающее функционал регистрации пользователей, создания и управления событиями, управления приглашениями, а также отслеживания участников. Приложение должно обеспечивать удобный и безопасный доступ пользователей к основным функциям.

## Необходимый функционал

### Пользователи

- Регистрация пользователя
- Аутентикация пользователя
- Подтверждения почтового адреса
- Восстановление пароля

### События

- Создание события
- Редактирование события
- Удаление (отмена) события
- Отслеживания количества зарегистрировавщихся пользователей
- Рассылка уведомлений

### Приглашение

- Запрос приглашения
- Отклонение приглашения
- Принятие приглашения
- Отмена приглашения

### Участник

- Просмотр участий
